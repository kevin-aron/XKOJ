2022年 桂林电子大学信息科技学院 创新训练项目。

使用Django，Celery+Redis作为消息队列，配合docker下的多进程完成测评任务，每个任务单独隔离，任务类型：①通过时限+编译指令完成程序编译；②文件对比。并未实现spj。
